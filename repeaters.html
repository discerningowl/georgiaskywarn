Local SKYWARN Nets (Not-Linked)

These repeaters are activated by Local County SKYWARN coordinators or County EMA. These are not-linked to the state-wide repeater network. Local officials will pass weather reports on to WX4PTC or the NWS.

NWS Weather Radio Stations (NOAA Weather Radio)

These are NOAA Weather Radio transmitters operated by the National Weather Service. They broadcast continuous weather information, warnings, watches, and forecasts 24/7. You can receive these stations with a weather radio or scanner. Most stations transmit on one of seven VHF frequencies between 162.400 and 162.550 MHz.

CHIRP Import Instructions

How to Import into CHIRP

  1. Download Complete! Your CSV file has been saved to your downloads folder.
  2. Open CHIRP software on your computer.
  3. Connect your radio to your computer via USB cable.
  4. Download from radio: Radio ‚Üí Download From Radio (to read existing channels).
  5. Import CSV: File ‚Üí Import ‚Üí Select the downloaded CSV file.
  6. Review channels: Check imported repeaters in the channel list.
  7. Upload to radio: Radio ‚Üí Upload To Radio.

Note: The CSV includes all Georgia SKYWARN repeaters (linked and non-linked). Total: 58 repeaters.

RT Systems Import Instructions

How to Import into RT Systems

  1. Download Complete! Your CSV file has been saved to your downloads folder.
  2. Open RT Systems software for your radio model.
  3. Connect your radio to your computer via USB cable.
  4. Read from radio: Communication ‚Üí Read from radio (to load existing data).
  5. Import CSV: File ‚Üí Import ‚Üí CSV File ‚Üí Select the downloaded file.
  6. Select Bank 22: Skywarn when prompted for import location.
  7. Review repeaters: Navigate to Bank 22 to see all imported channels.
  8. Write to radio: Communication ‚Üí Write to radio.

Note: All 58 Georgia SKYWARN repeaters will be imported into Bank 22: Skywarn with channel numbers 00-57.
